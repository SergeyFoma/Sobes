Менеджер проекта IN компании. Нужно в кратчайшие сроки разработать проект 
интернет магазина электроники.

В проекте участвуют три разработчика. Необходимо подобрать наиболее оптимальный вариант
разработки при котором увеличится скорость разработки, тестирования и запуск продукта.
Применяемые технологии: Docer, Github, Njinx, HTML, CSS, Postgresql

Подготовь структуру проекта на Django для интернет магазина.
Разбей задачи на три логических блока. 

Опиши структуру каждого блока и преимущества такого подхода. Напиши код для каждого блока.

Описание должно быть кратким, без общих фраз.
